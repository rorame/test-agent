# Crossing-Time Agent

## Задание

> «Сколько понадобится времени гепарду, чтобы пересечь Москву-реку по Большому
> Каменному мосту?»

Агент отвечает на количественные вопросы о реальном мире: собирает факты,
рассуждает по шагам, считает кодом и проверяет собственный результат.

## Архитектура

- Ядро — [`deepagents`](https://github.com/langchain-ai/deepagents)
- Поиск — Tavily
- LLM — через OpenRouter
- Трейсинг — Langfuse
- Совместимость с протоколом A2A (Agent-to-Agent)

## Как запускать

```bash
make run
```
